  • 对象单体模式
    • 为了解决箭头函数this指向的问题 推出来一种写法 对象的单体模式 
            var person = {
          name:'tom',
           age:18,　　
          fav:function(){       
              console.log(this.name)
          }
        }
        和下面相同
        var person = {
          name:'tom',
          age:18,
          fav(){
               // 当前this指向了定义时所在的对象（window）
               console.log(this.name);
            }
          }
